## üõ†Ô∏è How to do it (Free & Easy)

You can use a free tool on your Windows computer to convert the `out` folder into an APK file.

### Step 1: Get the "Website 2 APK" Tool
1.  Search Google for **"Website 2 APK Builder"** (There is a free version available).
2.  Or use **"Web2Apk"** (online converters also work, but offline software is safer).
3.  Download and install it on your Laptop.

### Step 2: Configure the App
1.  Open **Website 2 APK Builder**.
2.  **Website Type:** Select **"Local HTML Website"**.
3.  **Directory:** Browse and select your `d:\ai\clock-masjid\app\out` folder.
4.  **App Title:** "Masjid Clock".
5.  **Package Name:** `com.masjid.clock`
6.  **Orientation:** Landscape.
7.  **Permissions:** Ensure "WAKE_LOCK" (Prevent Sleep) is checked.

### Step 3: Build & Install
1.  Click **"Generate APK"**.
2.  It will create a file like `MasjidClock.apk`.
3.  Copy this file to your **USB Drive**.
4.  Plug the USB into the TV.
5.  Use a File Manager on the TV to click and install the APK.
